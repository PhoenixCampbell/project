Smarter Faculty–Course Matching

Faculty rate courses they’d like to teach. The system uses optimization to suggest balanced, preference-based assignments.

Get Started

How It Works

1. Faculty Rate Courses

Faculty select a term and rate each course 0–4 to express their interest.

2. Optimization Solver

The backend formats these preferences for Google OR-Tools and computes optimal matches.

3. Review Results

Admins can download Excel summaries of course–faculty assignments.

Key Features

Secure Login

Supports Microsoft SSO or department-level passwords.

Preference Grid

Simple and fast 0–4 scale for entering faculty preferences.

Excel Export

Download optimized results for quick departmental review.
